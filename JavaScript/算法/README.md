### 算法

> 冒泡排序：数组中相邻两个数做比较，再将较大的数和下一个数再做比较，最后将大数放在数组右侧；

> 选择排序：寻找最小数的过程，将所有小的依次放在数组右侧

> 插入排序(折半插入)：新建空数组，取原数组中的每一个数取出，再在新数组中比较放入对应位置

> 快速排序：选择基准数，通常将素组分为左右两队，再通过递归再对两队数组进行同样的操作进行排序
